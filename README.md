# NOTEFY

NOTEFY is a multifunctional notepad organizer for personal events that offers a variety of features, including:

- Events List
- Notes List
- Text Editor
- Speech-to-Text
- Digital Clock
- Image-to-Text Converter
- Dark and Light Modes
- Ringtone Picker
